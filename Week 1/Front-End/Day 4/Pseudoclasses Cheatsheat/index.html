PseudoClasses

:hover, :active, :focus for interactive elements

:hover - apply styles when the user's mouse is hovering over an element.
Hover Me! 
                                    
    HTML:
        <span class="text__hover">Hover me!</span>    
    CSS:
        .text__hover:hover {
            color: #6fd2ff;
        }   
                                    
                                
                                    
    HTML:
        <button class="button__hover">Hover me too!</button>    
    CSS:
        .button__hover:hover {
            background-color: #41b14f;
        }
:active - apply styles when the user is actively interacting with an element, such as clicking on a button.
Click Me! 
                                    
    HTML:
        <a href="#" class="link__active">Click Me!</a>    
    CSS:
        .link__active:active {
            text-decoration: none;
            color: #e5e5e5;
        }   
                                    
                                
                                    
    HTML:
        <button class="button__active">Click me too!</button>    
    CSS:
        .button__active:active {
            outline: 4px dotted #DC143C;
            background-color: #41b14f;
        }
:focus - targets an element when it has focus, such as when a user clicks on an input field. 
                                    
    HTML:
        <input type="text" id="input__focus" class="input__focus">    
    CSS:
        .input__focus:focus {
            background-color: rgb(15, 180, 255);
            outline: 2px solid #41b14f;
        }

:link, :visited, :target for link states

:link - targets unvisited links in order to apply styles.
A Link 
                                    
    HTML:
        <a href="#" class="link">A Link<a>     
    CSS:
        .link:link {
            text-decoration: none;
            color: #0fb4ff;
        }
Another Link 
                                    
    HTML:
        <a href="#" class="link">Another Link<a>    
    CSS:
        .link:link {
            text-decoration: none;
            color: #0fb4ff;
        }
:visited - targets links already visited by the user in order to apply styles.
A Link 
                                    
    HTML:
        <a href="#top" class="visited-link1">A Link<a>     
    CSS:
        .visited-link1:visited {
            color: #41b14f;
        }
Another Link 
                                    
    HTML:
        <a href="#top" class="visited-link2">Another Link<a>     
    CSS:
        .visited-link2:visited {
            color: #e5e5e5;
        }
:targeted - targets sections of code marked by 'id' and uses '#' anchor links in order to apply styles.
Link to :link 
                                    
    HTML above in :link section (must set id='link' so we can refer to it as a target): 
        <span id="link">:link - targets unvisited links in order to apply styles.</span>    
    
    HTML:
        <a href="#link">Link to :link<a>     
    CSS:
        #link:target {
            background-color: #ff0000;
        }
Link to :active 
                                    
    HTML above in :active section (must set id='active' so we can refer to it as a target): 
        <span id="active">:active - apply styles when the user is actively interacting with an element, such as clicking on a button.</span>    
    HTML:
        <a href="#active">Link to :active<a>     
    CSS:
        #active:target {
            background-color: #ffff00;    
        }

:enabled, :disabled, and :checked for form control states

:enabled - targets enabled form elements. 
                                    
    HTML:
        <input type="text" id="enabled-input" class="text-input">    
    CSS:
        .text-input:enabled {
            border: 2px solid #ffff00;
        }

        .text-input:enabled:focus {
            outline: 2px solid #ffff00;
        }
:disabled - targets disabled form elements. 
                                    
    HTML:
        <input type="text" id="disabled-input" class="text-input">    
    CSS:
        .text-input:disabled {
            border: 2px solid #ff0000;
        }
:checked - targets checked form elements, like checkboxes and radio buttons.

   

                                    
    HTML:
        <input type="checkbox" id="checkbox1"><label for="checkbox1"></label>   
    CSS:
        input[type=checkbox]:checked + label {
            color: #41b14f;
            border: 1px solid red;
        }
   
                                    
    HTML:        
        <input type="checkbox" id="checkbox2"><label for="checkbox2"></label>    
    CSS:
        input[type=checkbox]:checked + label {
            color: #41b14f;
            border: 1px solid red;
        }

:required, :optional, and :valid/:invalid for form validation

:required - targets form elements with the required attribute. 
                                    
    HTML:        
        <input type="text" id="required-field" class="text-input" required>          
    CSS:
        .text-input:required {
            border: 2px solid #800080;
            background-color: rgb(65, 177, 79, .5);
        }

        .text-input:required:focus {
            outline: 2px solid #800080;
        }
:optional - targets form elements without the required attribute. 
                                    
    HTML:        
        <input type="text" id="optional-field" class="text-input optional">          
    CSS:
        .text-input.optional:optional {
            border: 2px solid #ff69b4 ;
        }

        .text-input.optional:optional:focus {
            outline: 2px solid #ff69b4 ;
        }
:valid/invalid - target form elements based on their validation status. 
                                    
    HTML:        
        <input type="email" id="email">          
    CSS:
        input[type=email]:invalid {
            outline: 2px solid #ff0000;
        }

:first-child, :last-child, :nth-child(), and :nth-last-child() for child elements

:first-child - targets the first element among a group of sibling elements.
:last-child - targets the last element among a group of sibling elements.
  1. Item 1 (first child)
  2. Item 2
  3. Item 3
  4. Item 4 (last child)
                                    
    HTML:        
        <ol class="firstlast">
            <li>Item 1 (first child)</li>   
            <li>Item 2</li>
            <li>Item 3</li>
            <li>Item 4 (last child)</li>    
        </ol>          
    CSS:
        .firstlast li:first-child {
            background-color: rgb(255, 255, 0, .5);   
        }

        .firstlast li:last-child {
            background-color: rgb(255, 0, 0, .5);   
        }
:nth-child - targets elements based on their position among a group of siblings.
:nth-last-child - targets elements based on their position among a group of siblings but counts from the end.
  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4
  5. Item 5
  6. Item 6
  7. Item 7
  8. Item 8
                                    
    HTML:        
        <ol class="nthchild">
            <li>Item 1</li>   
            <li>Item 2</li>
            <li>Item 3</li>
            <li>Item 4</li>
            <li>Item 5</li>   
            <li>Item 6</li>
            <li>Item 7</li>
            <li>Item 8</li>       
        </ol>          
    CSS:
        .nthchild li:nth-child(even) {
            border: 1px solid #ffff00;
        }

        .nthchild li:nth-child(3) {
            background-color: rgb(255,0,0,.5);
        }

        .nthchild li:nth-last-child(2) {
            background-color: rgb(65, 177, 79, .5);   
        }

:first-of-type, :last-of-type, :nth-of-type(), and :nth-last-of-type() for specific element types

:not for negation

:first-of-type - targets the first element of its type among its siblings.
:last-of-type - targets the last element of its type among its siblings.

This is the first paragraph

This is the second paragraph

This is the third paragraph

This is the fourth paragraph

This is the fifth paragraph

                                    
    HTML:        
        <div class="firstlasttype">
            <p>This is the first paragraph</p>
            <p>This is the second paragraph</p>    
            <p>This is the third paragraph</p>
            <p>This is the fourth paragraph</p>
            <p>This is the fifth paragraph</p>
        </div>
    CSS:
        .firstlasttype p:first-of-type {
            font-style: italic;
            text-decoration: underline;
            color: #41b14f;
        }

        .firstlasttype p:last-of-type {
            font-weight: 900;
            font-size: 2rem;
            color: #ff69b4;
        }
:nth-of-type - targets elements of a specific type based on a specified pattern. For example, to target every odd p element within a div.
:nth-last-of-type - similar to :nth-of-type(), but it counts from the last element of the specified type. For example, to target the second to last p element within a div.

This is the first paragraph

This is the second paragraph

This is the third paragraph

This is the fourth paragraph

This is the fifth paragraph

This is the sixth paragraph

This is the seventh paragraph

This is the eighth paragraph

This is the ninth paragraph

This is the tenth paragraph

                                    
    HTML:        
        <div class="nthlasttype">
            <p>This is the first paragraph</p>
            <p>This is the second paragraph</p>    
            <p>This is the third paragraph</p>
            <p>This is the fourth paragraph</p>
            <p>This is the fifth paragraph</p>
            <p>This is the sixth paragraph</p>
            <p>This is the seventh paragraph</p>    
            <p>This is the eighth paragraph</p>
            <p>This is the ninth paragraph</p>
            <p>This is the tenth paragraph</p>
        </div>         
    CSS:
        .nthlasttype p:nth-of-type(odd) {
            text-decoration: line-through;
        }

        .nthlasttype p:nth-of-type(-n + 3) {
            color: #ff69b4;
        }

        .nthlasttype p:nth-last-of-type(-n + 3) {
            color: #41b14f;
        }

        .nthlasttype p:nth-last-of-type(4) {
            font-weight: 700;
            font-style:italic;
            text-decoration: underline overline;
        }
:not() - used to apply styles to elements that do not match the specified selector. This can be particularly useful for excluding specific elements from a general style rule.

This is the first paragraph

This is the second paragraph

This is the third paragraph

This is the fourth paragraph

This is the fifth paragraph

This is the sixth paragraph

This is the seventh paragraph

This is the eighth paragraph

This is the ninth paragraph

This is the tenth paragraph

                                    
    HTML:        
        <div class="nthlasttype">
            <p>This is the first paragraph</p>
            <p class="no-border no-color no-underline">This is the second paragraph</p>    
            <p>This is the third paragraph</p>
            <p class="no-color no-underline">This is the fourth paragraph</p>
            <p class="no-underline">This is the fifth paragraph</p>
            <p class="no-border no-underline">This is the sixth paragraph</p>
            <p class="no-color">This is the seventh paragraph</p>    
            <p>This is the eighth paragraph</p>
            <p class="no-border">This is the ninth paragraph</p>
            <p class="no-underline">This is the tenth paragraph</p>
        </div>         
    CSS:
        .div-negation p:not(.no-border) {
            border: 1px solid #ff69b4;
        }

        .div-negation p:not(.no-color) {
            color: #41b14f;
        }

        .div-negation p:not(.no-underline) {
            text-decoration: underline;
        }
