## Math and Code Operators

**Exponentiation (Math):** 
### ${(x + 5)}^{y + 1}$

**Exponentiation (Python):** 
### `(x + 5)**(y + 1)`

---

**Floor Division (Math):** 
### $ \lfloor x / y \rfloor$

**Integer Division (Python):** 
### `x // y`

---

**Modulo (Math):** 
### $x \bmod{y}$

**Modulo (Python):** 
### `x % y`
